To Do - Home Page Detail

Having read the panel about course administration you might want to consider ways in which to make your course home page clear and comprehensive. Use the following checklist to decide whether you will provide the following detailed course information, either as a separate PDF or a separate page off the welcome page.

  • Course title, number, credits, prerequisites
  • Course outcomes
  • Course description
  • Course syllabus
  • Course pacing within course time frame is provided on the calendar
  • Schedule of class topics and assignments (including deliverables, due dates, instructions for submitting)
  • Standard marking policy
  • Course facilitator name, telephone, e-mail
  • Regularly scheduled office hours
  • Preferred methods to contact the course facilitator
  • Hardware requirements (connection speed, hardware, software, plugins)
  • Software skills with links to online tutorials
  • Course expectations, including participation
  • Course policies clearly identified regarding plagiarism, test taking, late work, copyright laws, and confidentiality with links to policies
  • Information on support services
  • Information on disability services
  • Comprehensive list of necessary materials is provided with information on how to obtain materials.

Course home page tips:

  • A course distinguishes itself administratively with a unique code. But the title is what students remember. To help their retention, repeat the course code and title in subsequent headings in the home page
  • Course information needs to appear on the course home page. It can also be “packaged” in a manner so that it is available for students to download and saved, preferably when they are in a free Wi-Fi zone. The main reason for making the course home page down-loadable is that students student’s can access key information from a remote location, where Wi-Fi is not always available.
  • Prompt students to become responsible for their own time management. Make milestones across a whole course explicit and then include these milestones as an iCal file for them to download to their phone calendars.
